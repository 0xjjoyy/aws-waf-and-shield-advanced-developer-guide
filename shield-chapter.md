# AWS Shield<a name="shield-chapter"></a>

AWS provides AWS Shield Standard and AWS Shield Advanced for protection against DDoS attacks\. AWS Shield Standard is automatically included at no extra cost beyond what you already pay for AWS WAF and your other AWS services\. For added protection against DDoS attacks, AWS offers AWS Shield Advanced\.
